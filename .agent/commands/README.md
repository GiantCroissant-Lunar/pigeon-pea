# Agent Commands

This directory contains custom commands that agents can execute.

## Purpose

Commands define specific actions or operations that can be invoked by agents to perform tasks.

## File Format

Commands can be defined as:

- Shell scripts (`.sh`)
- PowerShell scripts (`.ps1`)
- Python scripts (`.py`)
- YAML command definitions (`.yaml`)

## Example Commands

- Build and test automation
- Code generation utilities
- Development workflow helpers
- Project-specific operations
