# Agent Workflows

This directory contains workflow definitions that orchestrate multiple agent actions.

## Purpose

Workflows define sequences of operations, decision trees, and complex processes that agents can follow.

## File Format

Workflows can be defined as:

- `.yaml` - YAML workflow definitions
- `.json` - JSON workflow configurations
- `.md` - Documented workflow procedures

## Example Workflows

- CI/CD pipelines
- Code review processes
- Feature development flows
- Issue triage procedures
